# DHT11 Temperature & Humidity Monitor

## What it does
Reads temperature and humidity every 2 seconds.
Calculates heat index and prints all values to Serial Monitor.
Triggers a warning if temperature exceeds 30°C.

## Hardware
- Arduino Uno
- DHT11 sensor
- Jumper wires

## Libraries
- DHT sensor library by Adafruit
- Adafruit Unified Sensor

## Output
--------------------
Temperature: 24.80 °C
Humidity: 52.00 %
Heat Index: 24.69 °C
--------------------
