# 02 - Button + LED

## What it does
Reads a push button connected to pin 2.
LED on pin 13 turns ON when button is pressed.
Prints "Button Pressed" to Serial Monitor while held.

## Hardware
- Arduino Uno
- Push button
- LED
- 10kΩ resistor
- Jumper wires

## Concepts Learned
- Digital input using digitalRead()
- Conditional logic in hardware (if/else)
- Serial Monitor for debugging
- Input vs output pin configuration\
